«Grokking Algorithms» գրքի հայերեն թարգմանությունը բարդ տեխնոլոգիական հասկացությունները մատուցում է պարզ, զվարճալի և տեսողականորեն հագեցած ձևով։ Այն հիանալի ուղեցույց է թե՛ սկսնակների, թե՛ փորձառու մասնագետների համար՝ հնարավորություն տալով յուրացնել ալգորիթմների տրամաբանությունը մայրենի լեզվով և նպաստելով հայկական ՏՏ համայնքի զարգացմանը։
